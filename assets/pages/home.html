What is Digital Lab?

We live in a highly digital world filled with technology, but it is easy to forget just how it actually is. Specifically, all computer systems, microcontrollers, and electronic devices use the concept of logic gates. This is what makes up the fundamental decision-makers, connecting the physical electronics world to the digital software world. It is important to grasp all of these concepts from an early stage. However, one major problem students face when trying to visualize all of these concepts is that it is simply not possible to look inside a physical digital logic gate to see exactly how it works. Students are simply left to try to correlate the operation of a circuit to its name and symbol. To address all these learning challenges, the Digital Lab project was created. This project aims to provide a better understanding of what a logic gate actually is and how it works. This project will use a user-friendly front-end built from a combination of HTML, CSS, JavaScript, and Bootstrap technologies to provide a dynamic platform for students to learn, visualize, and test what they know about logic gates.
