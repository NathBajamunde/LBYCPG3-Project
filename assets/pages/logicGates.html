Logic Gates

What are Logic Gates?

A Logic Gate is a digital circuit that allows or prevents signals from passing based on a specified condition. These signals represent two states: TRUE or FALSE. Usually, the positive supply voltage represents TRUE while 0V represents FALSE.

Categories of Logic Gates

Logic Gates are identified by the function they perform. Their symbols represent Boolean operations and allow electrical circuits to perform logical decision-making.

Basic Gates

Logic Gate Boolean Expression
NOT Gate X = A'
AND Gate X = A • B
OR Gate X = A + B

Universal Gates

Logic Gate Boolean Expression
NAND Gate X = (A • B)'
NOR Gate X = (A + B)'

Exclusive Gates

Logic Gate Boolean Expression
XOR Gate X = A ⊕ B
XNOR Gate X = A ⊙ B
